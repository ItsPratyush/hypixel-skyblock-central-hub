# Hypixel SkyBlock Hub

A simple and clean hub website that collects the most useful external tools and resources for **Hypixel SkyBlock** players in one place.

The goal of this project is to make it easier for players to quickly access commonly used SkyBlock websites without having to search for links repeatedly.

---

## Features

- Curated list of popular and trusted Hypixel SkyBlock resources
- Clean and responsive UI built with Tailwind CSS
- External links open in new tabs so the hub always remains accessible
- Classic website visitor counter for basic view tracking
- Fully static site with no backend required

---

## Included Resources

- **SkyCrypt** – Player profiles, skills, net worth, pets, and dungeon progress  
- **SkyBlock Fandom Wiki** – Community-maintained guides and information  
- **Hypixel Wiki** – Official Hypixel wiki (may be out of date for SkyBlock content)  
- **Hypixel SkyBlock Forums** – General discussion and community help  
- **SkyShards** – Attribute fusion calculator  
- **SkyMutations** – Garden and greenhouse calculator  
- **Elitebot.dev** – Farming leaderboards and profile statistics  
- **SkyCofl** – Auction House and Bazaar price tracking  

---

## Tech Stack

- HTML5
- Tailwind CSS (via CDN)
- Static hosting (GitHub Pages compatible)
